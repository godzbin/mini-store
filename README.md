# mini-store
nodejs + sqlite + vue 小型商品展示页

## 安装依赖
```
npm install

cd store-web

npm install
```
## 启动(先打包前端，后启动node)
```
cd store-web

npm run build

cd ../

npm run start
```

# 访问

目前使用的是 3000 端口


# 如果要调试前端

## 安装依赖
